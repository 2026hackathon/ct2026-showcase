## The problem

Your to-dos are scattered across N tools (chat / email / Jira / calendar / GitHub / terminal), and you also have to babysit your agents. Capture friction is high, so most things get forgotten in the gap between "meaning to note it" and "actually noting it"; by the time a reminder fires it's already too late; and every morning you have to open a pile of windows just to figure out what to do today.

## How it works

Turn the always-empty notch into a second screen — an always-on AI assistant: collapsed it occupies only the two wings without blocking content, and it expands on hover. **Capture → track → remind → review**, the whole loop lives in this little strip. Based on "your relationship to it," everything you have to worry about in a day is re-grouped into five non-overlapping categories: Todo / WorkItem / Calendar / Message / Agent.

## Highlights

- **Screenshot to to-do in seconds**: F2 to screenshot → AI visual parsing → a structured to-do in 3 seconds, with auto-judged priority, suggested deadline, and the original image attached.
- **All your workflows on one screen**: Today suggests a work order, Work aggregates Jira+PR, Calendar merges into one timeline, Inbox condenses email into a single sentence.
- **Agent session monitoring**: bring agent run status up to the notch — running → awaiting confirmation → done card pops up, with a jump back to Terminal / iTerm / tmux.
- **AI morning & evening briefs**: the morning brief plans today's work order around meetings; the evening brief reviews today and flags tomorrow's high-priority items.
- **MCP extension**: next step, open an MCP server so any agent / script / CI can write reminders into Next, making it the unified reminder outlet.
